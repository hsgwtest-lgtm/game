# SoftEvo7 変更履歴

## 変更概要

以下の4つの主要機能を追加・修正しました。

1. 個体別物理環境 (Per-creature COF) — 各生物が保存時の物理パラメータで走行可能に
2. UI/UX 改善 — 速度ボタンアイコン、視点切り替え、確認ダイアログの統一
3. 生物削除機能 — 生物閲覧画面から保存生物を削除可能に
4. 距離表示修正 — スタート地点を基準0とした相対距離表示

## 動作の仕組み

### 個体別物理環境 (Per-creature COF)

- 保存時: engine.js の COF → creatureSaveManager に cof として保存
- レース時: p.cof から perCOF を構築 → SoftBody._cof に設定
- 物理計算: this._cof.gravity / airDrag / bounce 等を使用
- 独自環境モード (デフォルト): 各生物が保存時の物理パラメータで走行
- 同一環境モード: 全生物が共通の raceCOF で走行
- ゴースト対戦 (engine.js): body._cof ?? COF で対戦相手の独自 COF、通常個体はグローバル COF を使用

### 確認ダイアログの統一

- index.html (saveUI.js) → showConfirm() → #modal-confirm
- viewer.html (viewer.js) → viewerConfirm() → #viewer-modal-confirm
- engine.js の switchToBuild() → window.showConfirm() → #modal-confirm
- すべて同一の UI パターン (Promise ベース、キャンセル/OK ボタン) を使用。
